Password generator

The password generator program is written in Python and creates strong, customizable passwords. It uses lists of alphabets, capital letters, numbers, and special characters to generate passwords based on user-defined length and complexity. Users can choose between "easy," "normal," and "hard" difficulty levels, each with different proportions of character types. The program ensures that the generated password meets the length and complexity requirements by filling in any remaining characters. Finally, it shuffles the characters to create a secure password and prints the result.

Coffee Machine

This Python program simulates a coffee machine that offers three types of drinks: latte, black coffee, and ice coffee. The program keeps track of available resources (water, coffee, milk) and calculates the cost of each drink. Users can place orders, check the availability of resources, and view the total profit made. The program ensures that sufficient resources are available before making a drink and handles payment transactions, including giving back change if necessary.
